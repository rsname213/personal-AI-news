    - filter_duplicates(articles, seen) returns (new_articles, dupe_articles) where new_articles are those whose normalized URL is NOT in seen
    - mark_as_seen(articles, seen) adds each article's normalized URL with today's ISO date to seen dict and returns it
    - save_seen_urls() writes the dict as indented JSON to SEEN_URLS_PATH
  </behavior>
  <action>
    Create `/workspaces/personal-AI-news/pipeline/deduplicate.py` with exactly these components:

    1. Module-level constants:
       - SEEN_URLS_PATH = os.path.join(os.path.dirname(os.path.dirname(__file__)), ".seen_urls")
       - DEDUP_WINDOW_DAYS = 7
       - _UTM_PARAMS = {"utm_source", "utm_medium", "utm_campaign", "utm_term", "utm_content"}

    2. _normalize_url(url: str) -> str — private function:
       - urlparse(url.lower().strip())
       - filter out UTM params from query string via parse_qsl
       - sort remaining query params with sorted() for stability
       - strip trailing slash from path (but keep "/" for root paths)
       - drop fragment
       - return normalized.geturl()

    3. load_seen_urls() -> dict[str, str]:
       - Returns {} if SEEN_URLS_PATH does not exist
       - Returns {} on json.JSONDecodeError or OSError (corrupt file — will be overwritten)
       - Otherwise returns json.load() result

    4. save_seen_urls(seen: dict[str, str]) -> None:
       - Writes seen to SEEN_URLS_PATH with json.dump(..., indent=2)

    5. purge_old_entries(seen: dict[str, str], window_days: int = DEDUP_WINDOW_DAYS) -> dict[str, str]:
       - Compute cutoff = today - timedelta(days=window_days)
       - Keep only entries where datetime.fromisoformat(date_str).date() >= cutoff
       - Use datetime.now(timezone.utc).date() for today

    6. filter_duplicates(articles, seen: dict[str, str]) -> tuple[list, list]:
       - Returns (new_articles, dupe_articles)
       - new_articles: articles whose _normalize_url(article.url) NOT in seen
       - dupe_articles: articles whose _normalize_url(article.url) IS in seen

    7. mark_as_seen(articles, seen: dict[str, str]) -> dict[str, str]:
       - today = datetime.now(timezone.utc).date().isoformat()
       - For each article, seen[_normalize_url(article.url)] = today
       - Return updated seen dict

    Imports needed: json, os, from datetime import datetime, timezone, timedelta, from urllib.parse import urlparse, urlencode, parse_qsl
